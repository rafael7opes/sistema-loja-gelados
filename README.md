# Sistema de Pedidos - Loja de Gelados

Projeto acadêmico simples desenvolvido em Python para o curso de Análise e Desenvolvimento de Sistemas da UNINTER. O código simula o caixa de uma loja de Açaí e Cupuaçu.

## O que aprendi e apliquei
* Uso de estruturas de repetição (`while`) para múltiplos pedidos.
* Uso de condicionais (`if/elif/else`) para regras de negócio (preços).
* Validação básica de entrada de dados do usuário.
* Formatação de valores com f-strings do Python 3.
